The Insurance Risk Premium Predictor:------------
    features column transformation before model training. The pipeline includes Random Forest classifier trained over a toy dataset. Model then exported in Pickle format for further prediction.

    The FastAPI is utilised for api connection between the model weights and  frontend.
    Since it is for base structure understanding, simple frontend layout is made using streamlit.
